## 📌 Git 설정 관련

### ✔️ Git 사용자 설정

| 명령어                | 설명                                                                                        |
| --------------------- | ------------------------------------------------------------------------------------------- |
| `git config --system` | Systme설정 파일(모든 시스템 사용자에게 적용)                                                |
| `git config --global` | 한 사용자의 Git Repository에 적용<br/>\* 예: `git config --global user.email "abc@abc.com"` |
| `git config --local`  | 하나의 Repository에만 적용                                                                  |
| `git config --list`   | 현재 설정 확인                                                                              |
